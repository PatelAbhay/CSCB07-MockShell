User Stories from Product Backlog that we will address in this sprint:
a) Prompt the user for input
b) As a user I should able to space part(s) of my command out   
   by more than 1 space of tab and still recive the appropiate  
   output for my input
c) As a user, I want to be able to quit/leave the shell/program
d) As a user, I want to be able to convert text to audio and  
   listen to the pronounciation of words
e) As a user, I want to be able to navigate across directories, 
   add directories, print my working directory and print all the 
   files inside a directory
f) As a user, I want to be able to look back at all past        
   commands that has been inputted or the last X number of       
   commands from the current session from oldest to most        
   recent order so that I can keep track of the changes done    
   in this session
